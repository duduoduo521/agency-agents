# 钉钉/飞书机器人集成开发计划

## 项目背景

The Code Agency 已经具备了AI驱动的自动化编码工作流，为了进一步提升用户体验和工作效率，需要集成钉钉和飞书机器人，实现企业级的消息通知和命令控制功能。

## 集成目标

1. **消息通知功能**：在任务开始、完成、失败等关键节点自动推送通知
2. **命令控制功能**：支持通过钉钉/飞书聊天命令触发工作流任务
3. **状态查询功能**：通过机器人查询任务状态和进度
4. **安全认证机制**：确保只有授权用户可以通过机器人触发任务

## 功能需求

### 1. 钉钉机器人集成

#### 消息接收与解析
- 支持接收 `/analyze [需求描述]` 命令
- 支持接收 `/design [需求 ID]` 命令
- 支持接收 `/code [设计 ID]` 命令
- 支持接收 `/test [项目路径]` 命令
- 支持接收 `/status [工作流 ID]` 命令
- 支持接收 `/deploy [环境]` 命令

#### 消息发送与通知
- 任务开始通知
- 任务进度更新通知
- 任务完成通知
- 任务失败通知

#### 安全机制
- 签名校验
- IP白名单
- 速率限制

### 2. 飞书机器人集成

#### 消息接收与解析
- 支持飞书特有的交互式卡片消息
- 支持相同的命令格式
- 支持富文本消息解析

#### 消息发送与通知
- 任务状态的交互式卡片展示
- 支持按钮交互，如"查看详情"

#### 安全机制
- 挑战模式验证
- 签名校验
- 速率限制

## 技术架构

### 整体架构
```
┌─────────────────┐    ┌──────────────────┐    ┌─────────────────┐
│   钉钉/飞书     │◄──►│  集成服务       │◄──►│  主服务        │
│   机器人        │    │  (独立进程)     │    │  (现有服务)    │
└─────────────────┘    └──────────────────┘    └─────────────────┘
                            ▲
                            │
                    ┌──────────────────┐
                    │  Web GUI        │
                    │  (控制面板)      │
                    └──────────────────┘
```

### 消息流向
1. 用户在钉钉/飞书中发送命令
2. 钉钉/飞书服务器向集成服务发送Webhook
3. 集成服务验证请求合法性
4. 集成服务调用主服务API
5. 主服务执行相应任务
6. 主服务通过事件系统通知集成服务
7. 集成服务向钉钉/飞书发送结果

### 管理端点
```
GET  /health                    # 健康检查
GET  /api/status               # 获取配置状态
POST /api/push-rules          # 更新推送规则
POST /api/test-notification   # 测试通知
```

## 安全考虑

1. **签名校验**：确保请求来自钉钉/飞书官方服务器
2. **IP白名单**：限制可访问的IP范围
3. **速率限制**：防止恶意刷请求
4. **权限控制**：区分普通用户和管理员权限
5. **数据加密**：敏感信息传输加密

## 部署方案

1. **独立部署**：机器人集成服务独立部署，端口3001
2. **容器化**：使用Docker容器部署
3. **负载均衡**：根据需要进行水平扩展
4. **监控告警**：实时监控服务状态

## 依赖关系

- 主服务（[server.js](file:///d:/agency-agents/server.js)）必须正常运行
- 需要配置钉钉/飞书机器人的Webhook地址
- 需要设置相应的Token和Secret

## 验证标准

1. 命令正确解析并执行对应任务
2. 通知及时准确推送到指定群组
3. 安全机制有效防止非法访问
4. 系统稳定运行无明显性能问题
5. 错误处理机制完善

## 风险评估

1. **安全风险**：Webhook可能被恶意利用，需严格实施安全措施
2. **性能风险**：大量并发请求可能导致服务不稳定
3. **网络风险**：网络延迟或中断影响消息传递
4. **兼容性风险**：钉钉/飞书API更新可能影响功能
